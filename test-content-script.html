üîß Technical Improvements

What Changed:

  • Content Script Injection: Sidebar is injected directly into web pages
  • No Side Panel API: Removed dependency on Chrome's side panel API
  • CSP Bypass: Content script runs in page context, avoiding iframe restrictions
  • Better Integration: Sidebar appears as part of the page, not separate panel
  • Improved UX: More natural split-screen experience like Edge browser

‚ö†Ô∏è Known Limitations

  • Browser Specific: Only works in Chrome/Chromium browsers
  • Page Injection: Sidebar is injected into each page (may affect some sites)
  • Z-Index: Some sites with high z-index elements might overlap sidebar
  • Performance: Content script runs on every page load
